# 🎓 數位教室工具箱 (Digital Classroom Toolbox)

> 提升課堂互動，讓教學變得更直觀、更有溫度。

這是一個專為教師設計的輕量化數位工具集合，旨在簡化課堂管理並增強與學生的互動。透過現代化的網頁技術，提供流暢且美觀的使用體驗。

---

## 1. 專案目的

本專案的核心目標是解決傳統課堂中「工具分散」與「介面枯燥」的問題。我們打造了一個整合式的平台，包含以下功能：

-   **⏱️ 全螢幕倒數計時器**：提供清晰的大型數字顯示，幫助學生有效管理時間，並在結束時自動觸發提醒。
-   **✨ 每日勵志金句**：隨機產生溫暖、正向的字句，為學生的學習生活注入活力與靈感。
-   **📢 現代感公告欄**：具備動態跑馬燈效果，用於呈現重要通知或今日學習重點，確保資訊傳達不遺漏。
-   **😊 每日心情簽到**：提供簡單直覺的互動介面，讓教師能即時掌握學生的情緒狀態，建立更好的親師生連結。

---

## 2. 實作步驟

本專案採用**模組化開發**與**原生技術優先**的原則，確保效能與易維護性：

### 第一步：基礎架構搭建
- 使用 **HTML5 Semantic Tags** 確保網頁結構清晰且符合 SEO。
- 採用 **CSS3 (Vanilla CSS)** 建立設計系統，包含：
    - 使用 CSS Variables 定義配色方案。
    - 運用 Glassmorphism（玻璃擬態）與 CSS Animations 提升視覺美感。

### 第二步：互動邏輯開發
- 使用 **Vanilla JavaScript** 處理各工具的核心邏輯：
    - `setInterval` 實現計時器精準倒數。
    - 陣列與隨機算法驅動金句產生器。
    - DOM 操作實現動態公告更新。

### 第三步：響應式設計優化
- 透過 **Flexbox** 與 **Grid Layout** 實現 RWD（響應式網頁設計），確保工具在電腦、平板與手機上都能完美運作。
- 整合 Google Fonts (Outfit & Noto Sans TC) 提供跨平台的一致字體美感。

---

## 3. 注意事項

-   **瀏覽器支援**：建議使用最新版本的 Google Chrome、Microsoft Edge 或 Safari 以獲得最佳的動畫效果。
-   **全螢幕體驗**：計時器工具在全螢幕模式下效果最佳（可按 F11 進入）。
-   **資料儲存**：目前工具均為前端實現，頁面重新整理後狀態（如心情簽到結果）將重置。
-   **自定義說明**：
    - 如需修改公告內容，請點擊跑馬燈下方的「設定」按鈕。
    - 如需更換金句，可編輯 `quote-generator/script.js` 中的語錄陣列。
